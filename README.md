
# Web Project Dashboard – Version 1.0

## Project Description

This project is a simple web dashboard created using HTML and CSS.
Version 1.0 is the initial and stable release.
It provides a basic structure that can be expanded in future versions.

---

## Version Details

* Project Version: 1.0
* Release Type: Baseline / Initial Release
* Status: Stable

---

## Technologies Used

* HTML5 – for webpage structure
* CSS3 – for styling and layout

---

## File Structure

* `index.html` – Main webpage file
* `README.md` – Project documentation file

---

## Page Layout Explanation

* The page has a header with the title **Web Project Dashboard**
* The main content is inside a centered container
* The container displays:

  * A welcome message
  * A short description of Version 1.0
* A comment is included in the code to add new features later
